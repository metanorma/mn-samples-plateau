=== 適用範囲

「データ更新」とは、既に3D都市モデルが存在する都市において、以下を行うことを指す。

* 1．データセットの追加
* 2．地物型の追加
* 3．地物の追加
* 4．空間属性の追加
* 5．主題属性の追加

「1．データセットの追加」は、既存のデータセットを全く使用せず、新しいデータセットを作成すること、つまり、新しい3D都市モデルに作り直すことであり、これを「全体更新」と呼ぶ。一方、「2．地物型の追加」から「5．主題属性の追加」までは、既存のデータセットの一部又は全部を残して新しいデータを統合することであり、これを「一部更新」と呼ぶ。

なお、標準製品仕様書の改定に伴いデータを変換すること（バージョンアップ）は本附属書におけるデータ更新に含まない。
