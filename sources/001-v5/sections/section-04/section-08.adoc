土地利用モデル（LOD1）では、土地利用の形状を面により表現する。土地利用モデル（LOD1）の取得イメージを<<tab-4-53>>に示す。

h| 説明
| 都市計画基礎調査の土地利用現況において作成された面に一致する。 +
高さは0とする。

|===

====== 土地利用モデル（LOD1）の定義

[cols="1a,^1a,1a,1a,3a,3a,2a"]
|===
| LOD | | 地物型 | 空間属性の型 | 取得基準 | 取得方法 | 補足
